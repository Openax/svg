# svg hosting

use them like this:

```markdown
![](https://raw.github.com/Openax/svg/master/down.svg?sanitize=true)
```

or like this:

```markdown
<img src="https://raw.github.com/Openax/svg/master/down.svg?sanitize=true"/>
```
